# 🌸 FLOIOIDE

**FLOIOIDE** est un jeu d'aventure et de plateforme développé en Python. Incarnez une fleur guerrière, gérez vos ressources en eau et en énergie, et terrassez des boss redoutables à travers trois biomes uniques : le Désert, la Forêt et la Ville.

## 🚀 Installation et Lancement Rapide

Le projet inclut des scripts d'automatisation qui créent un environnement virtuel (`.venv`), installent la bibliothèque `arcade` et lancent le jeu.

### Sur Windows :
Double-cliquez sur :
> `floioide.bat`

### Sur Linux / macOS :
Exécutez dans un terminal :
```bash
chmod +x floioide.sh
./floioide.sh
